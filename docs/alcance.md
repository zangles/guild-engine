# Problema
Falta de un lugar donde los gremios puedan realizar y programar actividades como raids, eventos, etc.
Tambien dar herramientas a los gremios para poder hacer un seguimiento de diferentes cosas: Donaciones, dkp, etc.
Hoy se utiliza discord para esto pero todo es trabajo manual o con bots que no son muy eficientes.

# Solucion propuesta
Una plataforma que centraliza estas herramientas en un solo lugar, para que los gremios puedan hacer estas actividades.
No pretende reemplazar discord, sino incorporarse a el con integraciones.
La plataforma permitiria a cualquier usuario registrado crear un nuevo gremio y aceptar miembros. Como buscar gremios ya existentes y solicitar unirse.
Un usuario puede pertenecer a mas de un gremio.
